Elementor Forms Setup

  1. 1

    Open your Elementor form

    Edit the page → click your form widget → go to Actions After Submit

  2. 2

    Add Webhook action

    Click + Add Action → select Webhook

  3. 3

    Set the URL

    Paste the endpoint URL above into the Webhook URL field

  4. 4

    Map your fields

    Set each field ID: name → Name field, phone → Phone field, address → Address field

  5. 5

    Add Authorization header

    Under Advanced → Headers add: Authorization = Bearer + your anon key

  6. 6

    Test it

    Submit a test entry. It should appear in your Leads tab within seconds.

⚠️ Elementor Tip

Requires Elementor Pro for webhook support. Free users can use the plugin Webhooks for Elementor from WordPress.org.

🥷

Ninja Forms Setup

  1. 1

    Install Webhooks add-on

    WordPress → Ninja Forms → Add-ons → install Webhooks

  2. 2

    Open your form

    Ninja Forms → Dashboard → click your form → Emails & Actions

  3. 3

    Add Webhook action

    Click + Add New Action → choose Webhook

  4. 4

    Configure the webhook

    Method: POST · Content Type: application/json · URL: paste endpoint above

  5. 5

    Add Authorization header

    Headers → Name: Authorization | Value: Bearer + your anon key

  6. 6

    Map field values

    Body params: name → name field, phone → phone field, address → address field, source → ninja

  7. 7

    Save and test

    Save the form. Submit a test entry. Lead appears in CRM instantly.

✅ Ninja Forms Advantage

Ninja Forms natively supports JSON body and custom headers — cleanest integration option.
